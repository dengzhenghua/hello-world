| - | - |
| url | [/hotSpotAnalysis/countSentimentAnalysiForMap](/hotSpotAnalysis/countSentimentAnalysiForMap) | 
| method | POST | 

#### 参数

| name | type | mandatory | 描述 | 
| - | - | - | - | - |
| programmeId | Long | Y | 方案ID | 

#### 返回值

| name | type | 描述 |
| :-: | :-: | :-: |
