## 1. 预警配置
### 1.1 监测创建 

| | |
| - | - |
| url | [/warningConfig/addWarningConfig](/warningConfig/addWarningConfig) | 
| method | POST | 

#### 参数

| name | type | mandatory | 描述 | 取值范围 |
| - | - | - | - | - |
| warningConfig.programmeId | long | Y | 所属方案 | 方案主键 |
| warningConfig.negative | long | Y | 负面情绪值超过数值  | 允许输入负数、正数，小数点后2位。 |
| warningConfig.positive | long | Y | 正面情绪值超过数值 | 允许输入负数、正数，小数点后2位。 |
| warningConfig.browseHeat | long | Y | 浏览热度超过数值 | 允许输入负数、正数，小数点后2位。 |
| warningConfig.hotNumber | long | Y | 达到相似问题热门数 | 允许输入负数、正数，小数点后2位。 |
| warningConfig.sensitiveWord | long | Y | 敏感词命中数 | 允许输入负数、正数，小数点后2位。 |
| sourceIds | long[] | N| 数据源 | 勾选 |
| sourceRegionIds | long[] | N| 来源区域 | 勾选 |


#### 返回值

| name | type | 描述 |
| :-: | :-: | :-: |
| programmeId | long | 方案Id |
| negative | string | 负面情绪值超过数值 |
| positive | long | 正面情绪值超过数值 |
| browseHeat | string | 浏览热度超过数值 |
| hotNumber | long | 达到相似问题热门数 |
| sensitiveWord | string | 敏感词命中数 |
| createDate | string | 创建时间 |
| updateDate | string | 修改时间 |


```javascript
{
    "success":true,
    "data":{
        "id":2,
        "createUser":"admin",
        "createDate":"2017-07-11 15:41:49",
        "updateUser":"admin",
        "updateDate":"2017-07-11 15:41:49",
        "sortField":null,
        "order":null,
        "tqmobile":null,
        "programmeId":1,
        "negative":-1,
        "positive":0,
        "browseHeat":null,
        "hotNumber":null,
        "sensitiveWord":null
    },
    "errorCode":"",
    "message":""
}
```
